Política de Privacidad de Datos

Última actualización: Abril 2025 (Esto para nada se ha escrito con ChatGPT)

Este sitio web, TodoLibro, ha sido desarrollado como parte de un proyecto final académico. No representa una empresa, no se encuentra en producción pública y no recopila ni almacena información personal con fines comerciales.

1. 📥 Información recopilada

Durante el uso de esta plataforma, se pueden procesar los siguientes datos únicamente de forma temporal y local: Correo electrónico, nombre de usuario y contraseña (solo para pruebas). Foto de perfil (Solo se almacena la ultima que has subido a nuestro servidor y se pueden utilizar URLs). Contenido como reseñas o listas de libros. Estos datos no se almacenan permanentemente ni se utilizan fuera del contexto de la demostración.

2. 🔒 Uso y almacenamiento de datos

Toda la información ingresada es utilizada exclusivamente para fines demostrativos. No se envía ni se comparte con terceros. La información puede eliminarse sin previo aviso.

3. 📤 Compartición de datos

No se realiza ningún tipo de compartición, venta ni cesión de datos a terceros.

4. 🍪 Cookies y seguimiento

Este sitio solo utiliza cookies para las sesiones. Este sitio web no utiliza funciones de rastreo.

5. 📧 Contacto

Este proyecto no cuenta con un dominio ni con un canal de contacto oficial. Para más información, por favor contacte directamente al autor/a del proyecto en el entorno académico correspondiente.

6. 🔄 Cambios

Dado que es un proyecto en desarrollo, esta política puede modificarse sin previo aviso como parte de ajustes funcionales o de presentación.
